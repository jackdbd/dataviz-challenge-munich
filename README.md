Built with:

- React
- D3
- vx

Clike [here](https://jackdbd.github.io/dataviz-challenge-munich) to explore the visualization!


# Installation

```
git clone git@github.com:jackdbd/dataviz-challenge-munich.git
cd dataviz-challenge-munich
yarn install
yarn start
```
